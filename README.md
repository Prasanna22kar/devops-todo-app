# 📝 DevOps Todo App

A simple Todo web application built with Flask and deployed using Docker & Docker Compose as part of a DevOps learning project.

## 🛠️ Tech Stack
- **App:** Python Flask
- **Containerization:** Docker, Docker Compose
- **CI/CD:** Jenkins (Phase 3)
- **Infrastructure:** Terraform + AWS EC2 (Phase 2)
- **Monitoring:** Prometheus + Grafana (Phase 4)
